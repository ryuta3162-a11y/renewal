# マシン画像をここに入れるだけ

**実装は完了しています。** このフォルダに画像ファイルを置いて GitHub に push すれば、自動で表示されます。

## やること（3ステップ）

1. LPの「マシンラインナップ」から画像を保存
2. この `machines/` フォルダに入れる
3. GitHub に push → Vercel が更新

拡張子は **.webp / .png / .jpg** どれでもOKです。

## ファイル名の例

| マシン名 | 入れるファイル名（どれか1つ） |
|---------|------------------------------|
| ランニングマシン | `treadmill.png` |
| リアデルト・ペックフライ | `rear-delt-pec-fly.png` |
| ラットプルダウン | `lat-pulldown.png` |

全部の対応表は `manifest.json` を見てください。

## LPの画像名が違う場合

`manifest.json` の `"file"` を LP の実際のファイル名に合わせて書き換えれば大丈夫です。

例：LPに `rear_delt.jpg` がある場合

```json
"リアデルト・ペックフライ": { "file": "rear_delt.jpg" }
```

## 注意

- 画像を入れないと「画像未登録」と表示されます（正常な動作です）
- 配置データ（図面上のマシン位置）はブラウザに別途保存され、画像追加で消えません
